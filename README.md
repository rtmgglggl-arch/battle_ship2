# 🚢 Морской бой (Telegram-бот)

Классический морской бой для двух игроков на разных устройствах. Каждый играет в своём личном чате с ботом; соединение устанавливается по коду приглашения.

## Поле и флот

- Размер поля: **10 × 10** клеток, координаты **A–J** по горизонтали и **1–10** по вертикали.
- Стандартный набор кораблей на каждого игрока:
  - 1 × 4-палубный (линкор)
  - 2 × 3-палубных (крейсеры)
  - 3 × 2-палубных (эсминца)
  - 4 × 1-палубных (катера)
- Всего: **10 кораблей, 20 занятых клеток**.

## Правила размещения

- Корабли ставятся либо **горизонтально**, либо **вертикально**.
- Корабли **не соприкасаются**, даже по диагонали — между ними минимум 1 пустая клетка.
- Расстановка генерируется автоматически; её можно перекинуть командой `/replace` до нажатия `/ready`.

## Механика боя

- Игрок называет координату, например `B7`, `A1`, `J10`.
- Возможные ответы:
  - 🌊 **Мимо** — ход переходит сопернику.
  - 🎯 **Ранил** — попал, но корабль ещё жив, ходишь снова.
  - 💥 **Убил** — корабль уничтожен, ходишь снова. Клетки вокруг убитого корабля автоматически помечаются как «мимо».
- Победа — когда у соперника потоплены все 10 кораблей.

## Игра на двух устройствах

1. Оба игрока открывают бота и отправляют `/start`.
2. Первый пишет `/new` — получает 6-символьный **код игры**.
3. Код передаётся сопернику любым способом (мессенджер, голос и т. п.).
4. Второй игрок пишет `/join КОД` в своём чате с ботом.
5. Каждый при желании перекидывает расстановку через `/replace`, затем подтверждает готовность через `/ready`.
6. Когда оба готовы — случайно выбирается первый ходящий, начинается бой.

## Команды

| Команда        | Что делает                                       |
|----------------|--------------------------------------------------|
| `/start`       | Приветствие и список команд                      |
| `/new`         | Создать игру, получить код приглашения           |
| `/join КОД`    | Присоединиться к игре по коду                    |
| `/replace`     | Перегенерировать случайную расстановку кораблей  |
| `/ready`       | Подтвердить готовность к бою                     |
| `/surrender`   | Сдаться и выйти из игры                          |
| `A1`, `B7`, …  | Ход во время боя                                 |

## Обозначения на поле

- `#` — твой корабль
- `*` — попадание (по твоему кораблю или по кораблю соперника)
- `O` — промах
- `.` — неизвестная / пустая клетка

После каждого хода бот присылает оба поля:

- 🎯 **Поле противника** — только твои выстрелы.
- 🚢 **Твоё поле** — твои корабли и выстрелы соперника по тебе.

## Запуск

Требуется Python 3.8+, `aiogram 2.x` и `Pillow`.

```bash
pip install aiogram==2.25.2
pip install pillow
```

1. Открыть [bot.py](bot.py) и подставить свой токен в `API_TOKEN` (получить у [@BotFather](https://t.me/BotFather)).
1. Запустить:

    ```bash
    python bot.py
    ```

1. Каждый игрок находит бота в Telegram и отправляет `/start` — дальше по инструкции выше.
